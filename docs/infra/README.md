# Infrastructure Docs (Dev)

## Что здесь описано
Документация по локальной dev-инфраструктуре проекта.

## Содержание
- `dev-postgres.md` — PostgreSQL в Docker Compose (развёртка, команды, нюансы)
- `env-and-gitignore.md` — как устроены `.env`, что коммитим, что игнорим

## Быстрый старт
Если нужно просто поднять Postgres:
1) См. `env-and-gitignore.md` → создать `infra/.env`
2) См. `dev-postgres.md` → команда запуска
